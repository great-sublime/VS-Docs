🎯 Goal:

Be comfortable writing scripts, basic web apps, and collaborating with version control.

  • Version control with Git and GitHub

  • Debugging and testing techniques

📌 Phase 2: Machine Learning & AI Foundations (Mid 2025 - End 2025)

🎯 Goal:

Be able to build, train, and evaluate machine learning models confidently.

  • Neural Networks and Deep Learning
  • Model Evaluation and Optimization
  • Using TensorFlow and PyTorch
